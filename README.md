JavaScript Calculator

Простой калькулятор, написанный на HTML, CSS и JavaScript без использования eval(). 

Поддерживает:

- базовые арифметические операции: +, -, *, /
- приоритет операций 
- отрицательные числа 
- защиту от деления на ноль
- смену темы (светлая/тёмная)
